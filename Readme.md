## 🧠 Core Techniques

These 5 mental tools replace all calculation:

```
1. Nibble Method          →  instant binary grouping (4-bit chunks)
2. Block Size Method      →  256 − mask = block, no math needed
3. Bucket Method          →  drop any IP into its range instantly
4. Anchor Method          →  known starting points for fast navigation
5. Wildcard Span Reading  →  read range directly, skip inversion
```

Full reference: [MENTAL_TRICKS.md](./MENTAL_TRICKS.md)
